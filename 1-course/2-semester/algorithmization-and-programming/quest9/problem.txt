1. База данных «Поликлиника»

Специализация: id, наименование специализации.
Диагноз: id, наименование диагноза.
Врач: id, ФИО, id специализации, категория (первая, вторая и т.д.).
Пациент: id, ФИО, год рождения.
Посещение: дата, id пациента, id врача, id диагноза.

Выборки из таблиц:
А) Пациенты, сгруппированные по врачу (внутри группы — по дате посещения).
Б) Пациенты, сгруппированные по диагнозу.
В) Пациенты, сгруппированные по дате посещения.
Г) Врачи, сгруппированные по специализации.

2. Банки

Клиенты: id, ФИО, номер счёта.
Операции: id, наименование операции (поступление, списание).
Движение денег: id клиента, id операции, дата операции, сумма.
